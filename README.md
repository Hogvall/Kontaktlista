# Kontaktlista
Individuell JS uppgift Kunskapskontroll #1 Kontaktlista applikation 




## Pedagogen i mig
För denna uppgiften så skulle vi skapa en kontakt lista. I rent lärande syfte fick jag då idén att ifall min kontaktlista ovanpå att den ska självklart uppfylla kraven för VG. Så vill jag att den ska skapa förståelse. För mig så vill jag då att min lista förklarar vad det är som händer.

Så just detta har jag skapat.
